# pycam-calib
python camera calibration software use opencv


## Instration

GitHubリポジトリからパッケージをインストールする

```bash
$ pip install git+https://github.com/DaiGuard/pycam_calib.git
```

バックグラウンドで使用する必要コマンドをインストールする

```bash
$ sudo apt install v4l-utils
```

## Usage

```bash
$ pycam_calib
```
